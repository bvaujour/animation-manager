# Tableau de bord centré sur une semaine

## Demande appliquée

- suppression du calendrier mensuel du tableau de bord ;
- suppression du choix du centre ;
- ajout d'un sélecteur de semaine identique aux autres pages ;
- calcul de toutes les informations pour l'ensemble des centres, du lundi au vendredi de la semaine choisie.

## Contenu du tableau de bord

Les indicateurs, l'état des centres et les alertes sont maintenant agrégés sur toute la semaine sélectionnée. Le détail quotidien reste affiché sous forme de cinq cartes, du lundi au vendredi.

Le sélecteur permet :

- de choisir une semaine enregistrée dans les périodes scolaires ;
- de passer à la semaine précédente ou suivante ;
- de revenir à la semaine actuelle.

Les liens vers le Planning ouvrent automatiquement la semaine affichée.

## Validation

- 14 tests ciblés réussis pour le tableau de bord, la navigation hebdomadaire et le récapitulatif ;
- contrôle Django réussi ;
- aucune migration nécessaire ;
- syntaxe Python, JavaScript et CSS validée.
