# Animateurs flottants — affectation directe

Dans Planning > Affectations, chaque lieu possède désormais une ligne « Animateurs flottants » avec une case par jour.

Deux usages sont possibles :

- glisser un animateur depuis la liste directement dans la case du jour ;
- sélectionner un animateur, puis cliquer sur la case « + Flottant ».

L'affectation est créée directement au niveau du lieu grâce au groupe technique invisible déjà utilisé. Aucune migration n'est nécessaire.

Les contrôles habituels restent appliqués : disponibilité du salarié et absence d'une autre affectation le même jour.
