# Navigation de semaine réorganisée

Le composant partagé par **Accueil** et **Planning** affiche désormais :

1. le nom de la semaine centré sur la première ligne ;
2. le bouton **Aujourd’hui** centré sur la seconde ligne ;
3. les flèches précédente et suivante de chaque côté du bouton **Aujourd’hui**.

Aucune migration de base de données n’est nécessaire.
