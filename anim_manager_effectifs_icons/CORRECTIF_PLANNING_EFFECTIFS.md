# Correctif Planning — Effectifs enfants

Le mode **Effectifs enfants** utilise désormais une mise en page dédiée :

- un centre occupe toute la largeur disponible ;
- les groupes d'un centre sont empilés verticalement ;
- les calendriers et tableaux FullCalendar sont forcés à 100 % de largeur ;
- aucun défilement horizontal n'est autorisé ;
- les cartes ne sont plus limitées par les anciennes règles `contain` et `overflow` ;
- les lignes d'effectifs peuvent revenir à la ligne sans comprimer leurs valeurs.

Le mode **Affectations** conserve sa mise en page actuelle.
