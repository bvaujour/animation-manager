# Qualifications proposées dans le remplissage automatique

Dans **Gestion > Qualifications**, chaque qualification possède maintenant
l'option **Proposer cette qualification dans le remplissage automatique**.

- activée : la qualification apparaît dans la fenêtre **Remplir auto** et peut
  être exigée pour un centre ;
- désactivée : elle reste disponible sur les fiches animateurs, mais n'apparaît
  plus parmi les exigences du remplissage automatique.

Les qualifications existantes sont activées par défaut lors de la migration.
